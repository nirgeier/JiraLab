# Jira Basics Hands-On Labs

- A collection of Hands-On Jira Labs covering Users, Managers, and Admins.
- Each lab is a standalone lab and does not require completing the previous labs.
- Labs are organized into three tracks: **User** (001–006), **Manager** (007–008), **Admin** (009–013).

## Pre-Requirements

- Access to a Jira Cloud instance (free tier is sufficient)
- A web browser (Chrome, Firefox, or Edge)
- An Atlassian account
- For Admin labs: Jira Administrator permissions
