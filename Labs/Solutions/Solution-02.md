## Solution 02 - Navigation & Dashboards

### Task 1: Keyboard Shortcut Challenge

Key shortcuts to remember:

| Action          | Shortcut     |
| --------------- | ------------ |
| Create issue    | ++c++        |
| Close dialog    | ++escape++   |
| Go to Dashboard | ++g++ ++d++  |
| Focus search    | ++slash++    |
| Shortcut help   | ++question++ |

### Task 2: Build a Personal Dashboard

1. **Dashboards** → **Create dashboard** → Name: `My Training Dashboard`
2. Click **Add gadget** → search and add each gadget
3. Configure Filter Results: use JQL `assignee = currentUser() AND status != Done`
4. Configure Pie Chart: Project = your project, Statistic Type = Priority
5. Drag gadgets to arrange in two columns

### Task 3: Star and Organize

- Star issues: Open the issue → click the ☆ icon (top right)
- Star project: **Projects** list → hover → click ☆
- Star filter: **Filters** → **View all filters** → click ☆ next to a filter
- Set default dashboard: **Dashboards** → click **⋯** on your dashboard → **Set as default**
