# Jira Hands-On Labs

---

Welcome to the Jira Hands-On Labs! This workspace contains a collection of self-contained labs designed to help you master Jira — from everyday user tasks to project management and full administration.

Labs are organized into three tracks:

- **User Track** (Labs 001–006) — Navigate Jira, create and manage issues, use boards and filters
- **Manager Track** (Labs 007–008) — Team oversight, reports, and metrics
- **Admin Track** (Labs 009–013) — Project configuration, workflows, permissions, and automation

Each lab is independent and can be completed in any order within its track.
