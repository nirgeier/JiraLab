## Task 06 - Boards & Sprints

### Objective

Plan, execute, and complete a full sprint.

---

??? question "Task 1: Sprint Planning"

    1. Ensure your backlog has at least 12 issues with story points
    2. Prioritize the backlog (drag to reorder)
    3. Create a new sprint
    4. Drag 6-8 issues into the sprint (target: 20 story points)
    5. Set a sprint goal: describe what you aim to achieve
    6. Start the sprint with a 1-week duration
    7. Take a screenshot of the sprint board

??? question "Task 2: Sprint Execution"

    1. Move 3 issues to "In Progress"
    2. Complete 4 issues (move to Done)
    3. Leave 2 issues in "To Do" (simulating incomplete work)
    4. Add a comment on at least 2 issues describing work done
    5. Complete the sprint
    6. Send incomplete issues to the backlog
    7. Review the auto-generated Sprint Report

??? question "Task 3: Board Configuration"

    1. Go to Board settings
    2. Add a quick filter: `Only Bugs` (JQL: `type = Bug`)
    3. Add a quick filter: `My Issues` (JQL: `assignee = currentUser()`)
    4. Configure swimlanes by Epic
    5. Set WIP limits: In Progress = 3
    6. Take a screenshot of the configured board
