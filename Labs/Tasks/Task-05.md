## Task 05 - Search & Filters

### Objective

Master JQL queries and create a set of reusable saved filters.

---

??? question "Task 1: JQL Query Challenge"

    Write and test JQL queries for each scenario:

    1. All issues assigned to you, sorted by priority (highest first)
    2. All Bugs created in the last 7 days
    3. All issues in "In Progress" status with no due date
    4. All Epics that have the label `auth`
    5. All issues updated today
    6. All High or Highest priority issues that are not Done
    7. All issues where the summary contains "login"
    8. All unassigned issues created more than 3 days ago

    Save your JQL queries in a text file.

??? question "Task 2: Create Saved Filters"

    Create and save these 4 filters:

    1. **My Open Work** — Your open issues sorted by priority
    2. **Team Bugs** — All open bugs in the project
    3. **Sprint Scope** — All issues in the current sprint
    4. **Overdue** — Issues past their due date that aren't Done

    Star all 4 filters.

??? question "Task 3: Dashboard Integration"

    1. Add your "My Open Work" filter to your dashboard as a **Filter Results** gadget
    2. Add your "Team Bugs" filter as a **Pie Chart** gadget (group by Priority)
    3. Set up an email subscription on the "Overdue" filter (daily at 9 AM)
    4. Share the "Sprint Scope" filter with your project team
