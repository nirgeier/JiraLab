## Task 02 - Navigation & Dashboards

### Objective

Master Jira navigation and create a personalized dashboard.

---

??? question "Task 1: Keyboard Shortcut Challenge"

    Complete these actions using **only keyboard shortcuts**:

    1. Open the Create Issue dialog ( ++c++ )
    2. Close it ( ++escape++ )
    3. Go to your Dashboard ( ++g++ then ++d++ )
    4. Focus the search bar ( ++slash++ )
    5. Open the keyboard shortcut help ( ++question++ )

    Record: How many shortcuts can you remember after 5 minutes?

??? question "Task 2: Build a Personal Dashboard"

    1. Create a new dashboard: `My Training Dashboard`
    2. Add these gadgets:
        - Assigned to Me
        - Activity Stream (scoped to your project)
        - Pie Chart (by Priority)
        - Filter Results (use a custom JQL filter)
    3. Arrange in a two-column layout
    4. Take a screenshot of the completed dashboard

??? question "Task 3: Star and Organize"

    1. Star at least 3 issues
    2. Star 1 project
    3. Star 1 filter
    4. Verify all appear in **Your work** → **Starred**
    5. Set your new dashboard as the default
