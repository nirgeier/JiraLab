## Task 01 - Jira Overview

### Objective

Explore a Jira instance and identify its key components.

---

??? question "Task 1: Identify Project Components"

    1. Log in to your Jira Cloud instance
    2. Navigate to **Projects** and list all available projects
    3. For each project, note:
        - Project key
        - Project type (Scrum/Kanban)
        - Number of issues
    4. Take a screenshot of the Projects page

??? question "Task 2: Explore Issue Types"

    1. Open any project
    2. Create one issue of each type: Epic, Story, Task, Bug
    3. For each issue:
        - Write a meaningful summary (not "test")
        - Set an appropriate priority
        - Add at least one label
    4. Document the issue keys you created

??? question "Task 3: Issue Hierarchy"

    1. Create an Epic: `Training Lab - Task 01`
    2. Create 2 Stories under the Epic
    3. Create 1 Sub-task under each Story
    4. Link one Story to a Bug using "relates to"
    5. Take a screenshot of the Epic showing its child issues
