## About this Repository

- A collection of **Hands-On Jira Labs** for learning project management with Atlassian Jira.
- Labs are organized into three tracks: **User**, **Manager**, and **Admin**.
- Each lab is a **standalone** module and does not require completing previous labs.

## Pre-Requirements

- Access to a Jira Cloud instance (free tier is sufficient for most labs)
- A web browser (Chrome, Firefox, or Edge recommended)
- An Atlassian account
- For Admin labs: **Jira Administrator** permissions on your instance

## How to Get a Free Jira Instance

1. Go to [atlassian.com/software/jira/free](https://www.atlassian.com/software/jira/free)
2. Click **Get it free**
3. Create an Atlassian account (or sign in)
4. Name your site (e.g., `yourname-training`)
5. You now have a fully functional Jira Cloud instance with up to 10 users
